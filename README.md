# PRODIGY_SD_05
## Web Scraping Application

### Internship: Prodigy Infotech
### Task-05

## Task Description
Create a program that extracts product information such as product name, price, and rating from an e-commerce website and stores the data in a CSV file.

## Website Used
https://books.toscrape.com

## Technologies Used
- Java
- Jsoup Library
- CSV File Handling

## Features
- Scrapes product name, price, and rating
- Stores extracted data in CSV format
- Simple and efficient implementation

## How to Run the Program
1. Open the project in IntelliJ IDEA
2. Add Jsoup library to the project
3. Run `WebScraper.java`
4. CSV file `products.csv` will be generated automatically

## Output
The program generates a CSV file containing scraped product details.
